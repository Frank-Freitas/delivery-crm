# delivery-crm

## Descrição
Sistema desktop para CRM e comandas offline de delivery de pizzas, implementado em JavaFX com Maven.

## Tecnologias
- Java 17
- JavaFX 17
- Maven 3.6+

## Pré-requisitos
1. JDK 17 instalado.
2. Maven configurado no PATH.


## Estrutura do projeto
```
crm/
├─ pom.xml
└─ src/
   └─ main/
      └─ java/
         └─ com/delivery/crm/
            └─ App.java
```
